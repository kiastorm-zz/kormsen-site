
This module provides color fill classes. These classes are mostly used for filling svg colors

```html
<div class="fill-white-washout">fill white washout</div>
<div class="fill-light-washout">fill light white washout</div>
<div class="fill-washout">fill washout</div>
<div class="fill-dark-washout">fill dark washout</div>

<div class="fill-extra-light-storm">fill extra light storm</div>
<div class="fill-light-storm">fill light storm</div>
<div class="fill-storm">fill storm</div>
<div class="fill-light-tarmac">fill dark storm</div>

<div class="fill-extra-light-tarmac">fill extra light tarmac</div>
<div class="fill-light-tarmac">fill extra light tarmac</div>
<div class="fill-tarmac">fill tarmac</div>
<div class="fill-dark-tarmac">fill dark tarmac</div>

<div class="fill-extra-light-sky-blue">fill extra light sky blue</div>
<div class="fill-light-sky-blue">fill light sky blue</div>
<div class="fill-sky-blue">fill sky blue</div>
<div class="fill-dark-sky-blue">fill dark sky blue</div>

<div class="fill-extra-light-turquoise">fill extra light turquoise</div>
<div class="fill-light-turquoise">fill light turquoise</div>
<div class="fill-turquoise">fill turquoise</div>
<div class="fill-dark-turquoise">fill dark turquoise</div>
```
